#마크다운

- 마크다운은 텍스트 기반의 마크업 언어
- 간단한 웹 기반 문서나 웹페이지 작성에 사용
- 마크다운은 html로 변환되어 웹에서 표시

##마크다운 기본문법

1. Header

#제목 (h1)
##부제목 (h2)
###소제목 (h3)

2. list

 - 순서없는 리스트
 - 목차 2
 - 목차 3

 1. 순서있는 리스트
 2. 순서 2
 3. 순서 3

3. 링크
 - 대괄호랑레 소괄호
 - [네이버](https://www.naver.com)

4. 이미지삽입
 - ![대체텍스트(설명텓스트)](이미지 주소)
 - ![사과](../img/다운로드.jpg)

 5. 인용구
 - 꺽쇠 기호 `>` 사용
 > 인용된내용

 6. 강조
 - 별표하나는 이탤릭체, 두개는 진하게
 - *이탤릭*
 - **굵게**

 7. 수평선

 8. 테이블
